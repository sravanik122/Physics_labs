# Measurement of Length Using Different Tools

## Input Parameters
- **Object Length** – Actual length of the object to be measured  
- **Measuring Tool Type** – Ruler / Vernier Caliper / Screw Gauge  
- **Least Count** – Smallest measurable value of the selected tool  
- **Zero Error** – Instrument zero offset  
- **Alignment Error** – Error due to improper alignment  
- **Parallax Error** – Error caused by viewing angle  
- **Unit System** – mm / cm / inch  
- **Temperature** – Ambient temperature affecting expansion  
- **Surface Roughness** – Surface quality of the object  
- **User Precision** – Skill level of the user  

## Output Parameters
- **Observed Length** – Raw measured length  
- **Corrected Length** – Length after zero-error correction  
- **Total Measurement Error** – Combined error from all sources  
- **Final Length** – Most accurate measured value  
- **Measurement Accuracy (%)** – Accuracy relative to actual length
